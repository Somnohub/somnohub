Intégrez le test STOP-BANG sur votre site

Vous êtes médecin, pharmacien, ou vous animez un site sur le sommeil ? Proposez à vos visiteurs d'évaluer leur risque d'apnée du sommeil en 2 minutes. Gratuit, sans inscription, une seule ligne de code.

Pourquoi l'ajouter à votre site ?
  • Un outil concret et utile pour vos patients, plutôt qu'un simple texte
  • Le questionnaire STOP-BANG est validé cliniquement et utilisé en pratique courante
  • Vos visiteurs restent chez vous : le test s'affiche dans votre page
  • Aucune donnée collectée : le score est calculé dans le navigateur du visiteur

Aperçu

Voici ce que verront vos visiteurs

Le code à copier

Collez ce code à l'endroit de votre page où le test doit apparaître :

<iframe src="https://somnohub.fr/widget/stop-bang.html"
        width="100%" height="440" style="border:0;max-width:580px"
        title="Test STOP-BANG — risque d'apnée du sommeil"
        loading="lazy"></iframe>

Compatible avec WordPress, Wix, Squarespace, Doctolib Site, ou tout site acceptant du HTML.

Hauteur automatique (optionnel)

Pour que le cadre s'ajuste tout seul à mesure que le visiteur avance dans le test, ajoutez ce petit script :

<script>
window.addEventListener('message', function (e) {
  if (e.data && e.data.somnohubHeight) {
    var f = document.querySelector('iframe[src*="somnohub.fr/widget"]');
    if (f) f.style.height = (e.data.somnohubHeight + 20) + 'px';
  }
});
</script>

Conditions d'utilisation

  • L'intégration est libre et gratuite, y compris sur un site professionnel
  • Merci de conserver la mention « Test proposé par SomnoHub » affichée sous le test
  • Ne modifiez pas les questions : le score ne serait plus celui du questionnaire validé
  • Le test est un outil de dépistage, il ne pose aucun diagnostic
